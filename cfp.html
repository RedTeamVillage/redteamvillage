What kind of talks / Topics do we want?:

The talks must be focused on Offensive cyber security operations, red teaming, blue teaming and other offensive subjects and research.

  • Innovative attack / defense strategies
  • Red Teaming
  • Blue Teaming
  • Software/Hardware exploitation
  • Infra security
  • Security monitoring
  • Web Application exploitation
  • Purple teaming
  • Offensive Cloud Security
  • Incident Response
  • Threat Intelligence
  • Malware, bots and C2
  • Security Automation
  • Threat Hunting

Everything Else - Above are some suggested topics that loosely align with the theme, but being ajn open community we consider all talk subjects related to hacking or offensive cyber security. If your talk doesn't fit in one of these topics don't worry, the suggested themes are just a starting point.
Red Team Village doesn’t place any restrictions on topic. However, we requests something specific as well as general. We want new, creative and cool content with respect to offensive cyber security.
